# Intro

Hace unos meses supe de la app "Habla Quechua" de PromPerú para iOS y Android pero ¿Y los que no tienen o no usan un smartphone con uno de estos sistemas operativos?.

De la applicacion se ha extraído toda la información (texto, imagen y audio). El contenido de las frases están en archivos rar (en realidad son bases de datos sqlite) que se descargan la primera vez que entras a una sección. En cuanto a los nombres y el diccionario, son peticiones individuales de un archivo mp3.
